# موقع أبو فؤاد لخدمات تفصيل الخزائن وأعمال النجارة 🪚

[![Website Status](https://img.shields.io/website?url=https%3A%2F%2Fwww.njartafsel-khazaen.com)](https://www.njartafsel-khazaen.com)
[![GitHub Pages](https://img.shields.io/badge/Hosted_on-GitHub_Pages-blue?logo=github)](https://pages.github.com/)

موقع تعريفي وخدمي مخصص لعرض خدمات "أبو فؤاد" لأعمال النجارة وتفصيل الخزائن الخشبية في مدينة الرياض. يهدف الموقع إلى استقطاب العملاء المحليين من خلال تحسين محركات البحث (Local SEO) وتوفير تجربة مستخدم سريعة وسهلة لتصفح الخدمات والتواصل.

## 🔗 الروابط الرسمية (Live Links)
* **الموقع الأساسي:** [njartafsel-khazaen.com](https://www.njartafsel-khazaen.com)
* **المدونة:** [blog.njartafsel-khazaen.com](https://blog.njartafsel-khazaen.com)

## 🎯 مميزات المشروع (Features)
* **تصميم متجاوب (Responsive Design):** يتوافق مع جميع الشاشات (أجهزة مكتبية، أجهزة لوحية، وهواتف ذكية).
* **تهيئة محركات البحث (SEO Optimized):** مهيأ بالكامل للظهور في نتائج البحث المحلية في الرياض للكلمات المفتاحية الخاصة بالنجارة وتفصيل الخزائن.
* **أداء عالي (High Performance):** استضافة سريعة ومستقرة عبر GitHub Pages.
* **مدونة فرعية:** لتعزيز المحتوى والكلمات المفتاحية، مستضافة على Blogger ومربوطة بنطاق فرعي (Subdomain).
* **تكامل البريد الإلكتروني:** إعدادات DNS مجهزة لربط البريد الإلكتروني الاحترافي عبر Hostinger.

## 🛠️ التقنيات المستخدمة (Tech Stack)
* **واجهة المستخدم (Frontend):** HTML5, CSS3, JavaScript.
* **الاستضافة (Hosting):** GitHub Pages (للموقع الأساسي) & Blogger (للمدونة).
* **إدارة النطاق (Domain Management):** GoDaddy (إدارة سجلات A, CNAME, TXT, MX).
